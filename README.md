# Easy Dreambooth UI for Windows
 Based on Joe Penna Implementation of Dreambooth (https://arxiv.org/abs/2208.12242) by way of Textual Inversion (https://arxiv.org/abs/2208.01618) for Stable Diffusion (https://arxiv.org/abs/2112.10752). Tweaks focused on training faces, objects, and styles. This is an easy installer and UI for Windows and Nvidia Cards
